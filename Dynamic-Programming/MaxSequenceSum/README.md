# Maximum Sequence Sum

**Given:** array of N numbers

**Required:** find a sequence of contiguous numbers with max sum

**Example:**

* Input : 
        ``````````````````````````````````````````````````````
        31 | -41 | _59_ | 26 | -53 | 58 | _97_ | -93 | -23 | 84
        ``````````````````````````````````````````````````````
* Output : 
            ``````````````````````````````````
            Total : 187 | Start : 2 | End : 6
            ``````````````````````````````````
